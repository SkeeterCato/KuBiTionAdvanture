# 武器属性表

数据来源：`src/data_item.js` 中 `ITEM_DATA` 的 `type: 'weapon'` 条目。

说明：`ps` 是体力，`san` 是精神，`hp` 是生命。表内伤害为基础伤害，实战还会受技能、前缀、耐久和特殊逻辑影响。

## 战斗规则说明

- 射程会影响命中。玩家射程来自武器射程、敏捷技能和装备敏捷加成；怪物射程来自怪物数据。
- 界面里的“射程优势”不是直接命中率。玩家射程高于怪物时，玩家基本命中，怪物命中会降低；玩家射程低于怪物时，玩家命中会降低，怪物基本命中。
- 差距换算公式是 `abs(玩家射程 - 怪物射程) / (20 + abs(玩家射程 - 怪物射程))`。
- 攻击会扣本次消耗，增加武器耐久损耗。远程武器如果有 `bullet`，还会消耗 1 个对应弹药。
- `durableSaveData` 按物品 ID 记录耐久损耗，同名武器共用一份耐久状态；制造同名武器只增加数量，不会重置该 ID 的耐久损耗。
- `deadStaff` 按背包尸体数量加伤，最多按 1000 个尸体计算。
