# 药剂配方、效果与原料

数据来源：

- 药剂定义：`src/data_item.js` 中 `ITEM_DATA` 的 `type: 'poizon'` 条目
- 炼金配方：`src/data_studio.js` 的 `ALCHEMY_DATA`
- 状态名称：`src/data.js` 的 `STATE_DATA`
- 交易价值：`src/main.js` 的 `getValue`

规则备注：

- 本表只列药剂。`ALCHEMY_DATA` 中的肥料、水、魂和猛毒爆弹不是 `type: 'poizon'`，不列入表格。
- 交易价值优先使用 `ITEM_DATA[id].value`；没有直接价值时，按正向状态收益计算，其中 `ps` 按 0.5 倍，`temp` 和负数效果不计入。
- 炼金时间使用配方里的 `timeNeed` 数值，单位沿用游戏内部时间。
